# Trabalho Final: Exploração das estruturas de dados nativas da linguagem Python
---

Esta atividade versa sobre as Collections da linguagem Python: Container datatypes. Python possui containers embutidos: `dict`, `list`, `set` e `tuple`. Além desses, Python 3 implementa containers especializados: `namedtuple`, `deque`, `ChainMap`, `Counter`, `OrderedDict`, `defaultDict`, `UserDict`, `UserList`, `UserString`.
